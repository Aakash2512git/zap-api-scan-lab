# ZAP API Scan Lab

This repo is used to manually run OWASP ZAP API scans against a provided endpoint.

## How to run
1. Go to **Actions**
2. Select **Manual OWASP ZAP API Scan (Lab)**
3. Click **Run workflow**
4. Enter an API endpoint
5. Download the ZAP report artifacts

⚠️ Do not scan production endpoints without approval.
